## tailwind.css

- 필수 익스텐션  
  1.Tailwind CSS IntelliSense : 입력하면 속셩별로 자동완성  
  2.Tailwind CSS Explorer : 속성 찾기  
  3.Headwind : 속셩별로 재배열 (협업시 용이)
